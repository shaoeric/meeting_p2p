# Checklist

## 房间管理
- [x] 主持人可创建无密码房间，生成唯一房间号并展示
- [x] 主持人可创建有密码房间，生成唯一房间号并展示
- [x] 其他用户输入正确房间号+密码可成功加入房间
- [x] 输入不存在的房间号时提示"房间不存在"
- [x] 输入正确房间号但错误密码时提示"密码错误，无法加入房间"
- [x] 主持人点击"结束会议"后所有参会者被踢出，房间销毁
- [x] 房间内最后一人离开后房间自动销毁，房间号失效
- [x] 已销毁房间的房间号无法再次使用加入

## 用户显示名
- [x] 创建房间对话框包含"显示名"输入框（可选）
- [x] 加入房间对话框包含"显示名"输入框（可选）
- [x] 不填显示名时系统自动生成短 UUID（如 8 位十六进制）
- [x] peer-info 消息携带显示名并广播
- [x] 参会者列表展示用户的显示名而非原始 peerId

## P2P 语音通话
- [x] 多人在同一房间内可实时语音通话（全双工）
- [x] 用户退出房间后，其他用户不再听到该用户声音
- [x] 麦克风静音/取消静音功能正常

## 屏幕共享
- [x] 用户可选择屏幕/窗口进行共享，其他参会者看到实时画面
- [x] 同一时间仅允许一人共享屏幕，第二人尝试共享时收到提示
- [x] 共享者停止共享后，所有参会者画面消失，标注清空
- [x] 屏幕共享启动后标注工具栏可用

## 屏幕标注
- [x] 矩形标注：拖拽绘制，实时同步到所有参会者
- [x] 箭头标注：拖拽绘制（带箭头头部），实时同步到所有参会者
- [x] 文本标注：点击放置并输入文本，实时同步到所有参会者
- [x] 颜色选择器可调节标注颜色
- [x] 粗细滑块可调节矩形/箭头线条粗细
- [x] 字号滑块可调节文本大小
- [x] 撤销按钮可撤销最近标注，所有参会者同步
- [x] 重绘按钮可恢复被撤销的标注，所有参会者同步
- [x] 清空按钮可清除所有标注，所有参会者同步

## 跨平台
- [x] 应用在 Windows 上正常运行（构建成功，release/win-unpacked/ 可执行）
- [ ] 应用在 macOS 上正常运行（需 macOS 环境验证）

## P2P 架构
- [x] P2P 信令层基于 Trystero/BitTorrent tracker（无中心服务器）
- [x] STUN 服务器配置用于 NAT 穿透
- [ ] 同一局域网内 P2P 直连通信正常（见 local-p2p-testing 规范）
- [ ] 不同网络环境下可通过 NAT 穿透建立 P2P 连接（需多机多网环境验证）
- [x] 应用不依赖任何中心服务器运行（无服务端代码）
